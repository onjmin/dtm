# DTM アプリ概要

## 概要
MML中間言語を使用したピアノロール打ち込みモジュール（WebベースのDTM）

## 主要ファイル

| ファイル | 役割 |
|---------|------|
| `src/index.ts` | エントリポイント、エクスポート |
| `src/piano-roll.ts` | ピアノロールUI管理、ノート編集操作 |
| `src/renderer.ts` | Canvas描画（グリッド、鍵盤、ノート） |
| `src/mml-core.ts` | ノートデータ管理、MML生成 |
| `src/mml-parser.ts` | （現在未使用） |
| `src/linked-list.ts` | Undo/Redo用双方向リスト |
| `src/types.ts` | 型定義 |

## 主要機能

- **4トラック構成**: メロディー・サブメロ・ベース・伴奏
  - メロディー: 同時に1音、主旋律
  - サブメロ: 同時に1音、2小節以上間隔がある音符群
  - ベース: 同時に1音、全体的に音が低い
  - 伴奏: 同時に2音以上同時発音
- **第5トラック**: ドラム（プリセットパターン選択）
- **音階**: C1〜C8（104鍵、MIDIノート番号24-127）
- **ノート長**: 4分/8分/16分/32分音符 + 各3連符（16モード）
- **ズーム**: x1/x2/x4（縦横対応）
- **操作**: クリックでノート追加、ドラッグで移動/リサイズ
- **音量設定**: 0〜100

## サウンド関連
- MML生成は`mml-core.ts`の`generateMML()`メソッドで担当
- 実際の再生には外部ライブラリ`@kazuprog/mml-player`が必要
- **マスタ音量**: 全トラックに一律適用される音量調整（gainNode経由）
- **MIDI/MML読み込み**: 読み込み前に既存データを自動消去
